Twitch IRC Bot

IRC Chat Bot
Python, SQL, Twitch API, Raspberry Pi/ AWS

This Twitch IRC bot allows a streamer on the Twitch platform to create a bot to interact with users automatically on their channel.

The bot has a number of basic interactions and responses, as well other actions such as storing and loading quotes in a database and keeping track of each viewers' "points" to use for streamer-defined redemptions.

I currently have this set to run from my local Raspberry Pi, but it can be configured to run from a cloud environment as well.

ReddyReader

Web App
Ruby on Rails, Google Books API, Bootstrap, OAuth

ReddyReader uses Ruby on Rails and the Google Books API to allow a user to search for a book and receive a personalized estimation of how long it might take them to read it. Users can also log in through Goodreads and receive estimations for all books on their "shelf."

The app also delivers data such as comparative worldwide reading speed statistics and suggested future reading based on other popular user searches.
