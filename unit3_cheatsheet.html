A Cheatsheet for Ruby Integers, Strings, Arrays, and Hashes

INTEGER - a whole number.

Useful methods:
.chr - returns the character represented by the int's value
.even? - returns T/F if even or not
.gcd(n) - returns greatest common divisor of int and n
.lcm(n) - returns lowest common multiple of int and n
.times - iterates a block 'int' times
 STRING - a finite sequence of characters.

Useful methods:
.capitalize - capitalizes the first character in the string
.chomp - chops off return characters OR the passed characters
.downcase - downcase every character in the string
.each_char - performs a block on each character
.empty? - returns T/F if empty or not
.gsub(pattern, replacement) - replaces pattern in string with replacement
.include?(pattern) - return T/F if string includes pattern or not
.length - returns the length of the string
.reverse - returns the reversed string
.split(pattern) - splits a string at the optional pattern, and returns an array of those substrings
.upcase - returns an all uppercase version of the string ARRAY - an ordered collection of any object

.collect - returns a new array of the results of performing a given block on each element of the array
.delete(object) - deletes the given object from the array
.delete_at(index) - delete the object at the given index
.each - performs the given block on each element of the array
.include?(object) - returns T/F if the array includes the given object or not
.join - joins the elements of the array into a string
.length - returns the length of the array
.reverse - reverses the array
.transpose - transposes the rows and columns of the array HASH - Dictionary like collection of keys and paired values

.delete(key) - deletes the key/value pair from the hash
.each - performs the given block on each key/value pair in the hash
.has_key? - returns true if hash has the given key
.has_value? - returns true if hash has the given value
.length - returns the length of the hash
